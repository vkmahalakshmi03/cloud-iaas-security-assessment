## Frameworks and Benchmarks

- **CIS Benchmark for Microsoft Azure Foundations v2.0** — Center for Internet Security. Primary control reference for all Azure findings and hardening steps. https://www.cisecurity.org/benchmark/azure

- **CIS Benchmark for Google Cloud Platform v1.3** — Center for Internet Security. Primary control reference for all GCP findings and hardening steps. https://www.cisecurity.org/benchmark/google_cloud_computing_platform

- **NIST Cybersecurity Framework (CSF)** — National Institute of Standards and Technology. Used for risk categorization across Identify, Protect, Detect, Respond, and Recover functions. https://www.nist.gov/cyberframework

- **MITRE ATT&CK — Cloud Matrix** — Used for tactic and technique mapping on each finding to model attacker behavior against default configurations. https://attack.mitre.org/matrices/enterprise/cloud/
